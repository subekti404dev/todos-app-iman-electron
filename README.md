## How to Install
```bash
  # 1. clone this repo
        git clone https://github.com/subekti404dev/todos-app-iman-web.git
  
  # 2. move into project directory
       cd todos-app-iman
  
  # 3. install dependencies
       npm install
       # or
       yarn install

  # 4. running project
       npm run start
       # or
       yarn start 

  # 4. open project url on browser
       http://localhost:3000
      
```
