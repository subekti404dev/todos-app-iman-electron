## How to Install
```bash
  # 1. clone this repo
        git clone https://github.com/subekti404dev/todos-app-iman-electron.git
  
  # 2. move into project directory
       cd todos-app-iman
  
  # 3. install dependencies
       npm install
       # or
       yarn install

  # 4. running project
       npm run dev
       # or
       yarn dev 

  # 4. build project
       npm run make
       # or
       yarn make 
      
```
